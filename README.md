# 🚗 Online Car Rental System

## 📌 Abstract

The Online Car Rental System is a comprehensive, secure, and user-centric web application developed using the Python Flask framework. It modernizes the traditional car rental process by providing a seamless digital platform for both customers and administrators.

The system allows users to browse available vehicles, view detailed specifications, and make reservations . It also includes advanced backend logic for dynamic pricing based on demand, holidays, and customer loyalty discounts.



## 📌 Description

This project is designed to simplify and automate the process of renting vehicles. Customers can easily search, filter, and book cars, while administrators can manage inventory, users, and bookings through a powerful dashboard.

## ⚙️ Installation & Setup

1. Clone or download the project
2. Open the project folder
3. Install required dependencies:

   
   pip install flask
   
4. Run the application:

   
   python app.py
   
5. Open browser and go to:

  
   http://127.0.0.1:5000/
